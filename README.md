#  Lean Study  Lean 4

Repositório dedicado ao estudo de métodos formais, lógica matemática e demonstração de teoremas utilizando o **Lean 4**. 
Os códigos e exercícios aqui presentes fazem parte das reuniões do projeto de extensão e do acompanhamento do livro *Mathematics in Lean* (MIL).

---

##  Objetivo do Repositório
O propósito deste espaço é registrar a minha evolução no aprendizado do Lean 4, documentando resoluções de problemas do
Lean Game Server e do Mathematican in Lean principalmente.
No futuro espero poder contribuir na construção de um material didático em português de qualidade.

---

## 🛠️ Como rodar o projeto localmente

1. Certifique-se de ter o **Lean 4** e o **elan** instalados na sua máquina.
2. Certifique-se de ter a extensão do Lean instalada no **VS Code**.
3. Abra a pasta `EstudosLean` no seu VS Code.
4. O Lean Infoview abrirá automaticamente ao lado para mostrar o estado das metas (`Goals`).
